### Customization

#### Change Slider Backgrounds
Edit the gradient colors in `index.html` within slider slides:
```html
<div class="slide-bg" style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);">
```

#### Change Slider Text
Edit the slide titles and taglines:
```html
<h2 class="slide-title">Your Title Here</h2>
<p class="slide-tagline">Your tagline here</p>
```

#### Adjust Slider Speed
Modify the auto-slide interval in `script.js`:
```javascript
setInterval(() => {
    this.nextSlide();
}, 5000); // 5000ms = 5 seconds
```

#### Change Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #1e40af;
    --secondary-color: #0ea5e9;
    --accent-color: #f59e0b;
    /* ... */
}
```

#### Update WhatsApp Number
Replace `919876543210` with your WhatsApp number in all links.

#### Add Programs/Products
Duplicate the `.product-card` block in the Products section and customize content.

#### Update Contact Information
Edit the Contact section with your actual details:
- Address
- Phone number
- Email
- WhatsApp number
